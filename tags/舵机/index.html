TD8120MG舵机使用Arduino驱动

2021-07-10
    • TD8120MG
    • 舵机
    • Arduino

    最近闲着没事，想把宿舍的自动开门从小程序蓝牙改成WiFi的，于是想起了这块被闲置已久的ESP8266，使用Blinker控制，然而，在使用Ardunio库的时候，始终无法正确驱动TD8120MG舵机转动正确的角度。

    Talk is cheap,show you the code.

    期望在2秒内完成旋转，试验如下：

    方案①

    无法正确旋转180度。查询资料，TD8120MG舵机的控制脉宽为0.5ms-2.5ms，修改代码。

    方案②

    无法正确旋转180度。

    再次阅读资料，发现TD8120MG舵机的PWM频率是50Hz，再次修改代码。

    正确方案

    之前不知道在哪里看到有人说，控制舵机只需要脉宽达到即可，通过这次解决问题，可见并不是所有舵机都是只要求脉宽，起码TD8120MG会要求频率。
